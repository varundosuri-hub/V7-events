# V7 Events & Productions — Website Bundle

A fully self-contained, offline-ready luxury event management website.

## Contents
- `index.html` — single-page site (Hero, About, Services, Portfolio, Testimonials, Contact, Footer)
- `style.css` — all theme, layout, and animations (dark + gold, Playfair Display + Poppins)
- `script.js` — navbar, scroll progress, reveal-on-scroll, testimonials slider, contact-form → WhatsApp
- `images/` — 10 optimized WebP images (hero, about, 8 gallery)

## Run locally
Just open `index.html` in a browser. No build step, no server needed.

For full functionality (Google Fonts + Google Maps embed), an internet connection is recommended on first load. Everything else — images, styles, animations, WhatsApp deep-link — works offline.

## Customization
- Change the WhatsApp number in `script.js` (`WHATSAPP_NUMBER`) and in the floating button `href` in `index.html`.
- Replace any image in `images/` — keep the same filename to avoid editing HTML.
- Edit testimonials in the `TESTIMONIALS` array inside `script.js`.
